# Frontend Polish

Use this skill only for narrow visual refinement loops.

## Rules

- One visual note at a time.
- Keep edits localized.
- Re-check in the browser after each change.
- Do not broaden into architecture or large refactors.

## Good Uses

- spacing density
- hierarchy tuning
- motion softness
- alignment cleanup
- CTA emphasis adjustment
